smithay::backend

[−][src]Module smithay::backend::udev

Provides udev related functionality for automated device scanning.

This module mainly provides the UdevBackend, which constantly monitors available DRM devices and notifies a user supplied UdevHandler of any changes.

Additionally this contains some utility functions related to scanning.

See also anvil/src/udev.rs for pure hardware backed example of a compositor utilizing this backend.
