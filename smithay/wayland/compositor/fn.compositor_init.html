Create new wl_compositor and wl_subcompositor globals.

The globals are directly registered into the event loop, and this function returns a [CompositorToken] which you'll need access the data associated to the wl_surfaces.

It also returns the two global handles, in case you wish to remove these globals from the event loop in the future.
